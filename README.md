# LFS Checkout Test

A simple Python project for testing Git LFS checkout functionality.

## Running Tests

To run the tests:

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run tests:
   ```bash
   pytest
   ```

   Or with verbose output:
   ```bash
   pytest -v
   ```

## Project Structure

- `hello.py` - Simple Python script that prints "hello world"
- `test_hello.py` - Test file for hello.py
- `testdata/` - Directory containing test data files
